CONFIGURING THE SERVER

Español

On this page we will describe how you can set up your server to run mosat programs correctly.

  1. You need to be sure that gcc, g++ and make are installed on your system.
  2. Create an user react with home directory /home/react, you can also run the script reactAddUser.sh from the script directory.
  3. Create a directory in the html root (tipically ⁄var⁄www⁄html) called react.
  4. Create a symbolic link from ⁄home⁄react⁄public_html to «html-root»⁄react.
  5. Install the source code in ⁄home⁄reacr⁄source using (subversion from sourceforge)
    You should have 4 directories in the mosat directory: config_files, docs, main and script
    and lib and include in the trunk directory, just one directory above.
