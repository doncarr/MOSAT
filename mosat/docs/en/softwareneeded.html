SOFTWARE DEPENDENCIES

Español

Software needed for running MOSAT

  1. g++
    To compile the source code for your computer (server)
  2. powerpc-linux-g++
    To compile the source code for the RTU (RTU's based on powerpc microprocessor)
  3. make
    To compile automatically all the source code the server's and the RTU's
  4. gnuplot
    To generate the performance graphics
  5. apache web server
    To make the reports avaliable for everybody on the network
  6. Text Editor
    To create and modify the configuration files
