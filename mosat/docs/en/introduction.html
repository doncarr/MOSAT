MOSAT:Trains Departure Monitoring System (MOnitoreo de SAlidas de Trenes in spanish)

MOSAT is a complete set of programas for monitoring a light rail train system departures, it gets information from inductive sensors on the rails and send it to RTU's on the stations that generate logfiles with the data that are readed for a server remotely to generate reports, statics and performance graphics everyday.

This system is very configurable and can be used in any light rail system just by making some changes, because it reades configuration files in plain text format that are very easy to write and modify at anytime, throgh this document we will describe how you can have MOSAT working the right way.
