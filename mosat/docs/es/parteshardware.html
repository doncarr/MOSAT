Hardware necesario para correr MOSAT

  1. Sensores inductivos:
    Dos para censar salidas y llegadas en una estación
  2. RTU:
    Unidad Terminal Remota (Remote Terminal Unit en inglés), una por estación
  3. SERVIDOR:
    Conectado a la misma red que el RTU para mantener la comunicación para obtener los logfiles y generar reportes (Con cualquier distribución de linux instalada)
