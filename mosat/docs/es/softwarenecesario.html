DEPENDENCIAS DE SOFTWARE

English

Software necesario para correr MOSAT

  1. g++
    Para compilar el código fuente para su computadora (servidor)
  2. powerpc-linux-g++
    Para compilar el código fuente para la RTU (La RTU esta basada en un microprocesador powerpc)
  3. make
    Para compilar automáticamente todo el código fuente, el del servidor y el de la RTU
  4. gnuplot
    Para generar gráficos de desempeño
  5. Servidor web apache
    Para que los reportes estén disponibles para todos en la red
  6. Editor de Texto
    Para crear y modificar los archivos de configuración
