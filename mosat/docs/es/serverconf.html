CONFIGURANDO EL SERVIDOR

English

En esta pagina describiremos como puede configurarse el servidor para correr los programas de mosat correctamente.

  1. Es necesario asegurarse de que gcc, g++ y make están instalados en el sistema.
  2. Después creamos un usuario react en el directorio home /home/react, también podemos ejecutar el script reactAddUser.sh que esta en el directorio script.
  3. Creamos un directorio en el html de root (comúnmente ⁄var⁄www⁄html) llamado react.
  4. Creamos una liga simbólica desde ⁄home⁄react⁄public_html a «html-root»⁄react.
  5. Instalamos el código fuente ⁄home⁄react⁄source usando (subversion de sourceforge)
    Debemos tener 4 directorios en el directorio mosat: config_files, docs, main y script
    además debemos tener lib e include en el directorio trunk, que esta un nivel mas abajo.
  6. Instalamos el compilador cruzado Sixnet PowerPC usando el script installppc.sh, es necesario hacer esto desde root y copiar todos los archivos .tar y .gz al mismo directorio.
